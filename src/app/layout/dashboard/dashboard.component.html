Project Introduction


This is a Single Page Web Application(SPA) that consumes an ExpressJS REST API that runs on the Node Server. We did all CURD operations using that REST API. Here we use a MongoDB - NoSQL database to store our records. Optionally we can also use an MSSQL DB too. For the authentication part we've used two steps API authentication with OAuth and for the website, we use Firebase(For Learning). We've used AWS S3 Bucket to store our images. For the admin section, we use Angular UI and a Free Admin Theme.

This is a very simple curd application on the node but our endeavor was to cover most of the wings that we at least need to start a production level development. So let's start the DEMO.

Used Technologies
  • NODE JS
  • Angular 2.0
  • ExpressJs
  • MongoDB
  • Firebase
  • Amazon Web Service
